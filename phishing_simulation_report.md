# Phishing Attack Simulation & Detection Report

## Objective
To understand phishing attacks and learn detection and prevention methods.

## Tools Used
- Kali Linux
- Manual phishing email template
- HTML fake login page

## Phishing Email Simulation
A fake email was created requesting urgent account verification.

## Landing Page
A fake login page was designed to demonstrate how attackers trick users.

## Red Flags Identified
- Urgent message
- Fake links
- Request for credentials

## Prevention Techniques
- User awareness
- Email verification
- Avoid suspicious links
- Enable multi-factor authentication

## Conclusion
This task helped in understanding phishing techniques and social engineering awareness.
